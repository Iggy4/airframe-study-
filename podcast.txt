SECTION ONE
AIRFRAME STUDY QUESTIONS

Question one.
Where should the Emergency Locator Transmitter be located?

Answer: As far aft as possible, but not in the vertical stabilizer.
Explanation: The tail section often survives crashes better.
Memory cue: Think of a movie plane crash where the tail is still intact and sticking out of the wreckage, still blinking. The tail survives.

Question two.

Answer: All can be required.
Explanation: Starter systems vary by aircraft.
Memory cue: Think of different car models needing different keys or push-start buttons. Different setups, same result.

Question three.

Answer: A megohmmeter.
Explanation: Used for insulation testing.
Memory cue: Mega equals huge. If resistance is huge, you need a mega tool. Mega-ohms, megger.

Question four.

Answer: The specific alloy used.
